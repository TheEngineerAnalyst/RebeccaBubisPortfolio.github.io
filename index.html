Rebecca Bubis Portfolio

Recent graduate with a BS in Industrial Engineering and currently pursuing a MS in Business Analytics. Feel free to explore what I've worked on by clicking on one of the projects below! For more information about myself, take a look at my LinkedIn.

Analyzing Consumer Sentiment and Trends in Amazon Reviews

This project involves an analysis of 25+ GB Amazon product and review data using Google Cloud Platform, ultimately delivering recommendations to improve product categorization and customer engagement based on review trends and metadata.

Bankruptcy Risk Prediction

This project involves developing an app to assess bankruptcy risk by industry, based on a logit model, identifying key industry-specific financial indicators and providing valuable insights for targeted risk assessments.

Early Disease Prediction

This project aims to identify early-stage disease patterns using unsupervised machine learning techniques in Python by clustering symptom-based medical records to uncover key patient groups at risk for chronic diseases and enable earlier intervention for improved healthcare outcomes.

Hospital Length of Stay Cost Forecasting

This project aims to identify key drivers and develop a predictive model using Python that forecasts hospitalization costs based on length of stay and patient-specific features.

User Insights in the C2C Fashion Ecosystem

This project combined exploratory data analysis using SQL in Google Cloud BigQuery and Vertex AI with a Tableau dashboard to identify underserved demographics and customer groups in the growing C2C fashion market, providing actionable insights to enhance engagement and drive sales.

How Performance Drives Market Moves

This project used Python for exploratory data analysis to investigate correlations between financial statement elements and stock prices, replicating hedge fund strategies to generate sustainable profits.

Data Driven Modeling and Prognostics of Battery Degradation

My undergraduate honors thesis involves predicting battery remaining useful life (RUL) using a CALCE dataset and features machine learning models to analyze battery degradation.
